# zensical_poc
Test of zensical framework

This project is a proof of concept experiment of the Zensical docs as code with SSG framework.

## Description

Bogus draft material generated for the original MkDocs and Material for MkDocs theme evaluation.

## Dependencies

Before using this project make sure you have:
- Python
- Git
- Visual Studio Code

## To use this project

Follow the [setup instructions](https://zensical.org/docs/get-started/) to set up a Python virtual environment and then install Zensical.

With each VSCode session, activate the virtual environment (the Terminal may do this for you or type `.venv\Scripts\activate`).

Use the live preview server. In Terminal, type `zensical serve` use the localhost:8000 port in the message.

> [!WARNING]
> Run this project locally only. The GitHub workflow to deploy to GitHub Pages will not work because MNIT Enterprise account does not allow GitHub Actions hosted runners.
